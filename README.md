### Usage
Clone this repository.
Run "npm install" from the root folder.
Run "bower install" from the "public" directory.
Run "node app.js" to launch the web server.

```sh
$ git clone [git-repo-url] diggit
$ cd diggit
$ npm install
$ cd public
$ bower install
$ cd ..
$ node app.js
```
